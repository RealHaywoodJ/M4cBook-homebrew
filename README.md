# M4cBook-homebrew
| Simple repo for settling up M4 MacBook |  | Includes AI apps &amp; SaaS/Dev Environments |

## What Gets Installed

### CLI Tools
- **GitHub CLI (gh)** - Command-line tool for GitHub operations

### Applications
- **Postman** - API development and testing platform
- **Replit** - Online IDE and code collaboration platform
- **Warp** - Modern terminal application
- **Visual Studio Code** - Code editor and development environment
- **GitHub Desktop** - Native Git and GitHub GUI application
